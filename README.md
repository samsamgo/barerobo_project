# barerobo_project
안녕하십니까 정말 반갑습니다. 이번에 베어로보틱스와 함께할 기회를 주신점 너무 감사합니다.

일하게 되는 것은 이제 막 취업 전선에 뛰어드는 열정이 넘치는 저에게 있어 커다란 행운과 특권이라고 생각하게 되어 
이 회사에 입사하기를 간절히 희망합니다.
